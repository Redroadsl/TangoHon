# 単語帳管理ツール（单词本管理工具）

用于管理和编辑日语单词表（CSV 格式）的桌面应用程序。

## 功能

- 浏览并切换文件夹内的 CSV 单词表文件
- 单词的添加、行内编辑（双击单元格）、删除
- 支持五列：汉字、假名、翻译、词性、短语
- 实时搜索过滤
- 按任意列排序
- 撤销（Ctrl+Z，最多 50 步）
- 编码选择（gbk / cp932 / utf-8 / euc-jp）
- 新建空白文件
- 未保存修改检测与提示

## 技术栈

- **语言:** Python 3.10+
- **依赖:** 仅使用 Python 标准库（无需安装第三方包）
  - `tkinter` — 图形界面
  - `csv` — CSV 读写
  - `os`, `copy` — 文件操作、深拷贝
- **运行环境:** 需支持日文/中文显示的系统

## CSV 文件格式

| 列 | 内容 |
|----|------|
| 1 | 日语汉字 |
| 2 | 日语假名 |
| 3 | 中文翻译 |
| 4 | 词性 |
| 5 | 短语/例句 |

编码可在应用内选择（默认 gbk）。

## 使用方法

```bash
cd 単語本
python tango_editor.py
```

- 左侧选择文件
- 双击单元格直接编辑（Enter 移到下一行，Tab 移到下一列）
- 工具栏或右键菜单进行添加 / 删除
- Ctrl+S 保存，Ctrl+Z 撤销
